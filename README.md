# University Students DataBase Management System Project
Scenario title: Student database management system

## Description
As a developer, you are responsible for designing a simple system for managing student information.

## Requirements
1. The system should allow users to add,search ,delete and show and save student information.
2. The system should be user-friendly and easy to navigate.
3. The system should be secure and prevent unauthorized access.

## Technologies used
1. Java language version 20
2. IDE: IntelliJ ultimate 
3. Database: no db external or internal , we use file txt and HashMap to store date
4. Maven build tool.

## Features
1. Add student information.
2. search student information.
3. delete student information.
4. show all student information.
5. save to file.
6. for exit the system.
7. for back to main menu.

## How to use the system
1. Choose an option from the main menu.
2. Enter the required information.
3. Click on the desired option.

## Conclusion
The system allows users to manage student information easily and efficiently.
